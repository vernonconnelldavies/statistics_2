This repository is a place to put the dataset (ST1.csv) with regard to university module 'Statistics 2.' The dataset is referring to hospital data and survival rates with regard to patients in Scotland. 
The 'navy' dataset was used in the module 'Statistics 2' in the networks assignment. It details a communication network of British ships during World War 2 by both radio and Morse code.

The 'skills_test_1.R' file is the submitted R script for the first skills test in this module. Its a text mining assignmnet using historic inauguration speeches of United States presidents.

Finally the 'skills_test_2.R' file is about hospital and patient survival data. It starts with basic logistic regression and then moves to multilevel models with hospital location as a random effect. The code then tests the model produced with predictions to see if it produces results that make sense intuitively. For example, a 72-year-old in Glasgow hospital for 12 days should have a worse survival probability than a 32-year-old in Livingston for 3 days. The model confirms that.
