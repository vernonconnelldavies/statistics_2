This repository is a place to put the dataset (ST1.csv) with regard to university module 'Statistics 2.' The dataset is referring to hospital data and survival rates with regard to patients in Scotland. 
The 'navy' dataset was used in the module 'Statistics 2' in the networks assignment. It details a communication network of British ships during World War 2 by both radio and Morse code.

Finally the 'skills_test_1.R' file is the submitted R script for the first skills test in this module. Its a text mining assignmnet using historic inauguration speeches of United States presidents.
